## 📦 Installation Guide

Use the proper package manager for your operating system.

### 🐧 Ubuntu Setup

Run the following commands:

```bash
sudo apt update

sudo apt install git -y

git clone <repository_url>

cd <repository_folder>

chmod +x *.sh

./script_name.sh
```

---

## ⚡ Command Explanation

| Command | Description |
|----------|-------------|
| `apt update` | Updates package repository |
| `apt install git` | Installs Git |
| `git clone` | Copies repository to local machine |
| `chmod +x` | Grants execute permission |
| `./script_name.sh` | Runs the bash script |
